# HF-simulation
The code of monte carlo method which aims to give the simulation of MuSEUM experiment; particle/nuclear physics experiment to measure the Hyperfine Splitting of Muonium in J-PARC. I use using ROOT and GEANT4. 
https://geant4.web.cern.ch/node/1 
https://root.cern/
https://g-2.kek.jp/gakusai/research_MuHFS.html

1. RF.cc
To calculate the TMmode of microware in the cavity. We use TM110 and TM210 mode to demo the RF respectively.
